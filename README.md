# 🚀 Dermage: Incremetando o Futuro (Team She++)

Este repositório contém a análise técnica e estratégica desenvolvida para o Datathon Dermage 2026. Nosso foco foi transformar dados transacionais em uma jornada de fidelidade sustentável.

## 📊 Principais Descobertas
* **Taxa de Recompra:** Apenas 23,17% dos clientes retornam.
* **Zona de Ouro:** Os primeiros 90 dias são cruciais para a retenção.
* **Barreira de Categoria:** O tempo médio de transição entre produtos (Photoage -> Improve) é de 136 dias.

## 💡 Soluções Propostas
1. **Régua de 80 Dias:** Intervenção antes do vale crítico de churn.
2. **Estratégia Next-Step:** Amostras inteligentes com gatilhos de desconto via QR Code (Dia 60 ao 90).
3. **Foco em Marketplaces:** Expansão em canais de alto ticket e retenção orgânica.

## 🛠️ Tecnologias Utilizadas
* Python (Pandas, Matplotlib, Seaborn)
* Segmentação RFM / Clustering (K-Means)
* Análise de Sazonalidade e Geográfica
